# Purpose

## Mission

**Transform raw research into compelling narratives in seconds.**

We eliminate the hours spent structuring and formatting presentations, letting professionals focus on what matters: their insights and message.

## Vision

A world where anyone can communicate complex ideas clearly and beautifully, regardless of design skills or presentation experience.

## Core User

**Business professionals** who:
- Conduct research and analysis
- Need to communicate findings to stakeholders
- Don't have time to craft presentations manually
- Want professional-quality output without design expertise

### User Personas

1. **The Analyst** - Has data and insights, needs to present to leadership
2. **The Consultant** - Compiles research, delivers to clients
3. **The Founder** - Has a vision, needs to pitch to investors
4. **The Researcher** - Has findings, needs to share with teams

## Key Value Propositions

| Value | Description |
|-------|-------------|
| **Speed** | From research to presentation in under 60 seconds |
| **Quality** | AI-generated narratives with professional structure |
| **Personalization** | Tailored to your company's voice and brand |
| **Accessibility** | Voice-first input means no keyboard required |
| **Simplicity** | 3-step flow: Input → Preview → Present |

## What We Don't Do

- **Not a slide deck tool** - We generate narratives, not static slides
- **Not a design tool** - AI handles styling automatically
- **Not collaborative editing** - Single-user workflow (for now)
- **Not a data visualization tool** - Focused on narrative, not charts

## Success Metrics

1. **Time to presentation**: < 60 seconds average
2. **Completion rate**: >80% of started narratives reach presentation
3. **Retry rate**: <20% need regeneration
4. **User satisfaction**: Quality of generated content

## Guiding Principles

1. **Voice-first, not voice-only** - Support multiple input methods
2. **AI-augmented, human-controlled** - Users can always edit
3. **Beautiful by default** - Every output looks professional
4. **Progressive disclosure** - Simple by default, powerful when needed
